  ## Installation
      git clone git@github.com:Erebus009/employee-tracker.git
      cd employee-tracker
      npm i
      node app
  
  
  ## Usage
  Using up and down arrows keys inside the command line interface you can pick many options to change the database in mySQL from  terminal.
  
  ## Tests
